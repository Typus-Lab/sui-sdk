## Getting Started

To get started, clone the
[sbv2-sui](https://github.com/switchboard-xyz/sbv2-sui) repository.

```bash
git clone https://github.com/switchboard-xyz/sbv2-sui
```

Then install the dependencies

```bash
cd sbv2-sui
pnpm install
```
